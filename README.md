# 🐶 PawFlix Crypto Hub

A fun multi-page web application that combines **dogs, movies, crypto tracking, and news** into one interactive and responsive website using **HTML, CSS, and Vanilla JavaScript**.

This project demonstrates frontend development skills including:
- Responsive web design
- API integration
- Semantic HTML structure
- Clean CSS styling
- Modular JavaScript (no frameworks)

---

## 🌐 Live Features

### 🐶 Dog Gallery
- Random dog images using **TheDogAPI**
- Click button to load new dogs

### 🎬 Movies Search
- Search movies using **OMDb API**
- Displays movie title, poster, year, and description

### 💰 Crypto Dashboard
- Live cryptocurrency data using **CoinAPI**
- Displays real-time price updates

### 📰 News Feed
- Latest news using **NewsAPI**
- Displays trending articles

---

## 🎨 Design Features

- Fully responsive layout (mobile + desktop)
- Clean modern UI
- Centered logo branding (200px)
- Welcome banners on all pages
- Responsive WebP images using `srcset`
- Hover animations on navigation buttons
- Consistent color theme (blue, white, dark gray)
